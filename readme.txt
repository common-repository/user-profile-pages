== Description ==

Adds a 'profile page' for each user, where they can type a bit about themselves. User avatars and names on post and comment pages will be clickable, to take visitors to the profile page.

Available in languages:
* English
* Spanish

== Installation ==

1. Upload the `user-profile-pages` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

You must have some form of 'pretty permalinks' enabled for the 'user profile' "page" link to work.
